# 🏢 현장 근태관리 시스템 사용자 매뉴얼

이 매뉴얼은 현장 근태관리 시스템을 사용하는 **[본사 관리자]**, **[현장 관리자(현장대리인)]**, **[근로자]** 각각의 역할에 맞춘 기능 및 사용 방법을 안내합니다.

현재 시스템 접속 주소(예시): `https://attendance-site-two.vercel.app`

---

## 1. 🔑 공통 사항: 로그인 방법

시스템에 접속하면 역할에 따라 다른 아이디와 비밀번호 규칙을 사용하여 로그인합니다.

- **본사 관리자 (HQ Admin)**
  - **아이디:** 부여받은 관리자 ID (예: `관리자`, `admin` 등)
  - **비밀번호:** 발급된 비밀번호 (예: `admin123`)
- **현장 관리자 (Site Manager)**
  - **아이디:** 소속 **현장명** (예: `미래빌딩`, `다원아파트` 등)
  - **비밀번호:** 본인 **휴대전화 번호 뒤 4자리**
- **근로자 (Worker)**
  - **아이디:** 본인의 **이름** (예: `홍길동`)
  - **비밀번호:** 본인 **휴대전화 번호 뒤 4자리**

> [!TIP]
> 💡 최초 로그인 후 비밀번호가 맞지 않거나 아이디가 기억나지 않을 경우 현장 관리자 또는 본사 관리자에게 문의해 주세요.

---

## 2. 👑 본사 관리자 (HQ Admin) 매뉴얼

본사 관리자는 전체 현장과 소속 근로자, 휴가/연차 상태를 총괄적으로 관리하는 최고 권한을 가집니다. 로그인 시 `/hq-admin` 페이지로 접속됩니다.

### 주요 기능 가이드

1. **대시보드 (Dashboard)**
   - 등록된 총 현장 수, 전체 근로자 수, 오늘 출근한 총인원을 한눈에 파악할 수 있습니다.
2. **현장 관리 (Sites)**
   - 신규 현장을 등록하고, 계약 기간이나 주소 등의 정보를 수정/삭제할 수 있습니다.
   - 현장별로 속한 "부서(조직)" 목록을 생성하고 관리합니다.
3. **현장대리인 관리 (Managers)**
   - 각 현장을 책임질 현장대리인(Site Manager)의 계정을 등록하고 현장에 배정합니다.
4. **QR코드 관리 (QR Management)**
   - 현장에 비치할 **출석체크용 QR 코드**를 등록하고 다운로드(인쇄용) 할 수 있습니다. 
   - 생성된 QR은 태블릿 등에 띄워두거나 프린트하여 출입구에 부착합니다.
5. **전체 근로자 통합 관리 (Workers)**
   - 전국의 모든 현장에 속한 근로자를 조회하고, 수정, 휴직/퇴사 처리(활성화 상태 토글)를 할 수 있습니다.
6. **통합 연차 현황 (Vacation Status / Requests)**
   - 모든 현장 근로자의 연차/휴가(병가, 반차 등) 신청 내역을 조회합니다.
   - 신청된 휴가를 승인 또는 반려 처리하여 근태에 반영합니다.

## 3. 👷‍♂️ 현장 관리자 (Site Manager) 매뉴얼

현장 관리자는 자신이 배정된 특정 현장의 인력과 근태를 중점적으로 관리합니다. 로그인 시 `/site-manager` 페이지로 접속됩니다.

### 주요 기능 가이드

1. **대시보드 (Dashboard)**
   - 우리 현장에 속한 총 근로자 수와 오늘(현재)까지 출근한 인원 현황을 즉시 확인합니다.
2. **근로자 관리 (Workers)**
   - **개별 등록:** 새로운 근로자를 건별로 시스템에 입력할 수 있습니다.
   - **엑셀 일괄 등록:** 대규모 인원이 투입될 경우, 엑셀 양식을 다운로드하여 정보를 채운 뒤 '일괄 업로드'로 간편하게 여러 명을 한 번에 등록합니다.
   - 근로자의 소속 부서나 직책, 연락처 등을 수정할 수 있습니다.
3. **근태/출석 현황 (Attendance)**
   - 우리 현장 소속 근로자들의 일자별 출석 기록을 달력과 표 형태로 확인합니다.
   - 근로자가 스마트폰을 안 가져왔거나 QR 인식이 불가한 경우, **수기 등록/수정** 버튼을 통해 직접 출근 처리를 해줄 수 있습니다.
   - 월별 **근태 보고서(PDF/Excel)** 를 다운로드하여 급여 정산 증빙 자료로 활용합니다.
4. **연차/휴가 현황 (Vacations)**
   - 현장 근로자가 신청한 휴가(반차, 연차, 조퇴 등)의 승인/반려 처리를 담당합니다.
   - 각 근로자의 남은 연차 잔여 일수 현황을 확인하여 업무 스케줄 조율에 참고합니다.

## 4. 📱 근로자 (Worker) 매뉴얼

근로자는 본인의 출석체크와 휴가 신청 등을 쉽고 간단하게 모바일 또는 PC로 이용할 수 있습니다. 로그인 시 `/worker` 페이지로 접속됩니다.

### 주요 기능 가이드

1. **출근하기 (QR 출근)**
   - 가장 핵심적인 기능입니다. 현장에 도착하면 "출근하기" 버튼을 눌러 카메라 브라우저 접근 권한을 허용한 뒤, **현장에 비치된 관리자용 QR코드**를 스캔합니다.
   - 정상 스캔 시 "정상적으로 출석 처리되었습니다" 문구와 함께 출근 시간이 기록됩니다.
2. **내 출근 현황 (Home/Dashboard)**
   - 캘린더를 통해 이번 달의 내 출근 날짜와 누적 조퇴/연차 기록을 한눈에 확인할 수 있습니다.
3. **휴가 / 연차 신청 (Vacation)**
   - '휴가 신청' 버튼을 통해 원하는 날짜와 사유(연차, 반차, 병가, 경조사 등)를 선택하여 신청 서류를 제출합니다.
   - 현장 관리자가 승인하면 해당 날짜에는 자동으로 '연차(혹은 해당 휴가)' 로그가 기록되어 출근하지 않아도 정상 휴가로 처리됩니다.
4. **남은 연차 갯수 조회**
   - 올해 부여받은 총연차와 사용한 연차를 비교하여 남은 휴가일수를 실시간으로 확인합니다.

## ❓ 자주 묻는 질문 (FAQ)

**Q. 로그인이 안 됩니다. 어떻게 하나요?**
A. 근로자와 현장 관리자는 기본적으로 전화번호 뒤 4자리를 비밀번호로 사용합니다. 번호를 변경했거나 오타가 계속 난다면 본사 또는 현장 대리인에게 초기화를 요청하세요.

**Q. 현장에 도착했는데 QR코드가 훼손되어 스캔이 안 됩니다.**
A. 당황하지 마시고 현장 대리인(Site Manager)에게 말씀드리면 관리자 권한으로 '수기 출석' 처리가 가능합니다.

**Q. 엑셀 업로드 시 계속 오류가 납니다.**
A. 현장 관리자가 근로자 엑셀 업로드를 할 때는 반드시 시스템에서 제공하는 **'엑셀 양식 다운로드'** 파일 포맷을 변경하지 않고 그대로 사용해야 인식 가능합니다. 특히 '전화번호' 항목이 빈칸이 아닌지 확인하세요.
